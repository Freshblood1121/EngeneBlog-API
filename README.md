============================================================================================================================

This project is a REST API for managing a news site. The API provides the ability to receive, add comments to news, and authorize users using JWT tokens.

## Functionality

- Get a list of news with the ability to filter by category, date or keywords.
- View individual news with full information about it, including title, text, publication date and author.
- Adding comments to the news with the username and comment text.
- Authorization of users using JWT tokens that can be used to access protected API resources.
